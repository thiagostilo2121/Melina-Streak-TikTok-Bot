1 - PROCESO DE INSTALACIÓN Y CONFIGURACIÓN
---------------------------------------------

1. Descargue el programa desde el repositorio de GitHub y abra el archivo install.py.

2. Abra el archivo Config.lnk (o simplemente Config) y seleccione la primer opción.

3. Ingrese el nombre EXACTO* (incluidas las mayúsculas, minúsculas y demás caracteres) que desea darle al perfil de Google Chrome que será utilizado.

4. Ingrese el nombre EXACTO* del usuario de Windows que aloja el perfil de Google Chrome.

5. Vuelva a abrir el archivo Config.

6. Ingrese en la segunda opción e indique el horario en formato 24hs (xx:xx) al cual se enviará el mensaje automático.

7. (PASO 6 E INGRESAR 3ER OPCIÓN) Ingrese el nombre que desee darle el archivo con el cual usted identificará a un usuario objetivo y luego ingrese el nombre EXACTO* que tiene el usuario objetivo en TikTok para que el programa logre identificarlo.

8. (PASO 6 E INGRESAR 4TA OPCIÓN) Ingrese el mensaje que desea enviar al usuario objetivo.

9. Abra el archivo Start e inicie sesión en TikTok al menos una vez (con haber iniciado una vez, no deberá hacerlo más).

10. USTED A FINALIZADO EL PROCESO DE INSTALACIÓN Y CONFIGURACIÓN DE "MELINA STREAK TIKTOK BOT". A continuación se especificarán algunos problemas que puede afrontar y como solucionarlos.

* Incluya mayúsculas, minúsculas y demás caracteres que se encuentren en el nombre del perfil.

2 - POSIBLES PROBLEMAS Y SUS SOLUCIONES
---------------------------------------------

1. "No se encontró la conversación con el usuario especificado." Esto se puede deber a que ha especificado erróneamente el nombre del usuario objetivo o también porque no se encuentra en su bandeja de mensajes principal. Si el problema es este, envíele un mensaje al usuario objetivo e inténtelo otra vez.
